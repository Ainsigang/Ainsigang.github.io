linux makefile文件

Makefile文件编写规则：
target : prerequisites…
command
target：要生成的文件名
prerequisites: 生成target所依赖的文件
command: 执行的命令
eg:
hello : hello.o #生成hello可执行文件依赖于hello.o
gcc -o hello hello.o #生成hello的命令
hello.o : hello.c #生成hello.o文件依赖于hello.c文件
gcc -c hello.c #编译生成hello.o文件
.PHONY : clean #定义一个伪目标文件用于清除执行过程中生成的中间文件
clean :
rm *.o #清除中间文件hello.o
